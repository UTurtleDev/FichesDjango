Les Modèles

  1. Création du modèle

    • Créer une classe dans modeles.py :
      class Serie(models.Model):
          name = models.CharField(max_length=200)
          slug = models.SlugField()
          seen_seasons = models.IntegerField(default=0)
          number_of_seasons = models.IntegerField(default=0)
          description = models.TextField(blank=True)
          cover = models.ImageField(upload_to=None, blank=True)
  2. Créer les migrations
    • Créer le fichier de migration :
      python manage.py makemigrations <nomApplication>
